About

Kranc is a Mathematica application which turns a tensorial description of a time dependent partial differential equation into a module for the Cactus Computational Toolkit.

Kranc is free software distributed under the GNU General Public License (GPL) (version 2) for unrestricted use. We request that you cite our 2006 Kranc paper if you use Kranc, or Kranc-generated code, in a scientific publication. Note that Kranc has evolved significantly since the version described in the paper, both in usability and functionality.

Examples and tutorials

We provide several example scripts which can be adapted to develop your own codes. The short script SimpleWave.m generates a 1D wave equation code. There is also a tutorial to guide you through the process of downloading Kranc, installing the required components into your Cactus tree, and generating your first Kranc thorn.

Documentation

The documentation is provided in LaTeX format in the Doc directory. Type "make" to make the PDF version, or "make KrancDocHTML" to make the HTML version.

Download

You can download Kranc in either zip or tar formats.

$ git clone git://github.com/ianhinder/Kranc

Users and publications

Codes based on Kranc have been used in 16 publications, mostly in the area of Numerical Relativity for simulating black holes.

Kranc has been used at

  • The Albert Einstein Institute
  • The Applied Mathematics Department of the University of Southampton
  • The Center for Gravitational Wave Physics at Pennsylvania State University
  • The Center for Relativistic Astrophysics at Georgia Institute of Technology
  • The Relativity and Gravitation group at the University of the Balearic Islands
  • The Center for Computational Technology at Louisiana State University

Related Software

  • Cactus: Kranc generates modules for the Cactus Computational Toolkit, an infrastructure for building high performance parallel codes. These codes can run simulations on anything from laptops to supercomputers.
  • Carpet: Carpet is a mesh refinement driver for Cactus. The code generated by Kranc is fully compatible with Carpet, meaning that Kranc-generated code can be run with both fixed and adaptive mesh refinement.
  • Mathematica: Mathematica is a powerful mathematics software system. Kranc is written in Mathematica and requires it in order to run. Mathematica is proprietary commercial software.
